# Documentacao do Colapso OS

Esta pasta concentra a documentacao do projeto fora do sistema embarcado na imagem.

Arquivos principais:

- `architecture.md`: visao geral do boot, kernel, shell e apps.
- `filesystem.md`: layout do disco e formato do diretorio.
- `testing.md`: como rodar a cadeia de validacao.

Estado atual:

- Shell interno `bash` em versao `0.1`.
- Kernel freestanding para `i386`.
- Filesystem simples baseado em tabela fixa de entradas.
